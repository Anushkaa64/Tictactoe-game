# ❌⭕ Tic Tac Toe

A classic Tic Tac Toe game built using HTML, CSS, and JavaScript — play with a friend right in your browser!

## ✨ Features

- 🎮 **Two player** mode (Player X vs Player O)
- 🏆 Detects and announces the **winner**
- 🤝 Detects a **draw** when board is full
- 🔄 **Restart button** to reset the game
- ✌️ **New game button** to start a new game

---

## 🛠️ Built With

| Technology | Purpose |
|---|---|
| HTML5 | Game structure |
| CSS3 | Styling and layout |
| JavaScript | Game logic and DOM manipulation |

---

## 💡 How It Works

1. Each cell listens for a **click event**
2. JavaScript tracks whose **turn** it is (X or O)
3. After every move, the game checks all **8 winning combinations**
4. If a winning pattern is found → announces the **winner**
5. If all cells filled and no winner → announces a **Draw**
6. Restart button **resets** the board and turn
